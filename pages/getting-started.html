เริ่มต้นใช้งาน

🟢 Beginner ⏱️ อ่าน ~5 นาที
📋 Prerequisite
✓ พื้นฐานการใช้คอมพิวเตอร์

ก่อนจะเขียนโค้ดได้ เราต้องเข้าใจภาพรวมและเตรียมเครื่องมือให้พร้อมก่อน

📋 สิ่งที่ต้องมีก่อนเริ่ม

🗂️ บัญชีที่ต้องสมัคร (ฟรีทั้งหมด)

เราจะใช้บริการ 3 อย่างที่ สมัครฟรี ได้ทั้งหมด:

บริการลิงก์สมัครใช้ทำอะไรแผนฟรี
🐙 GitHub github.com เก็บโค้ดบนคลาวด์ Unlimited public repos
▲ Vercel vercel.com เผยแพร่เว็บออนไลน์ Hobby plan ฟรี
⚡ Supabase supabase.com ฐานข้อมูล + สมาชิก Free tier 2 โปรเจกต์
💜 แนะนำ

ใช้อีเมลเดียวกันสมัครทั้ง 3 บริการ จะจำง่ายและจัดการง่ายกว่า สมัครด้วย Google Account ได้ทั้งหมด

💻 ซอฟต์แวร์ที่ต้องติดตั้ง

ซอฟต์แวร์คืออะไรทำไมต้องใช้
Node.js โปรแกรมสำหรับรันโค้ด JavaScript เว็บไซต์สมัยใหม่ต้องใช้ตัวนี้ในการพัฒนา
Git ระบบควบคุมเวอร์ชัน บันทึกประวัติการเปลี่ยนแปลงโค้ด
VS Code โปรแกรมเขียนโค้ด เครื่องมือยอดนิยมสำหรับเขียนโค้ด

📖 ดูวิธีติดตั้งละเอียดในบท 🔧 ติดตั้งเครื่องมือ

🔤 ศัพท์สำคัญที่ต้องรู้ก่อน

กดคำภาษาอังกฤษเพื่อดูคำแปลไทย:

คำศัพท์ความหมายเปรียบเทียบ
Frontendส่วนที่ผู้ใช้เห็นบนหน้าจอ🏠 หน้าบ้านที่แขกเห็น
Backendส่วนที่ทำงานเบื้องหลัง🍳 ห้องครัวที่ทำอาหาร
Databaseที่เก็บข้อมูล🗄️ ตู้เก็บเอกสาร
Hostingที่พักเว็บไซต์บนอินเทอร์เน็ต🏨 โรงแรมที่เว็บพักอยู่
Deployการเผยแพร่เว็บไซต์ขึ้นออนไลน์📦 ส่งสินค้าออกจากโรงงาน
Repositoryที่เก็บโค้ดทั้งโปรเจกต์📁 โฟลเดอร์โปรเจกต์ทั้งหมด
🧠 ทดสอบความเข้าใจ — กดที่การ์ดเพื่อเฉลย
Frontend
👆 กดเพื่อดูคำตอบ
🏠 ส่วนหน้าบ้าน — สิ่งที่ผู้ใช้เห็นและโต้ตอบได้บนหน้าจอ
Backend
👆 กดเพื่อดูคำตอบ
🍳 ส่วนหลังบ้าน — ประมวลผลข้อมูลเบื้องหลัง ที่ผู้ใช้ไม่เห็น
Deploy
👆 กดเพื่อดูคำตอบ
📦 การนำเว็บไซต์ขึ้นออนไลน์ — ให้คนอื่นเข้าถึงได้จริง
Repository
👆 กดเพื่อดูคำตอบ
📁 ที่เก็บโค้ดทั้งโปรเจกต์ — โฟลเดอร์ที่ Git ติดตาม
