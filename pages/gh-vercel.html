เชื่อม GitHub กับ Vercel

🟢 Beginner ⏱️ อ่าน ~5 นาที
📋 Prerequisite
✓ GitHub
✓ Vercel

เมื่อเชื่อม GitHub กับ Vercel แล้ว ทุกครั้งที่เรา Push โค้ด Vercel จะ auto-deploy ให้ทันที!

🔄 Workflow อัตโนมัติ

✏️ แก้โค้ดในเครื่อง
⬇️
📤 git push ไป GitHub
⬇️
▲ Vercel ตรวจพบอัตโนมัติ → เริ่ม Build
⬇️
✅ Build สำเร็จ → เว็บอัปเดตทันที!

📌 วิธีเชื่อม (ทำครั้งเดียว)

  1. สมัคร Vercel ด้วย GitHub (ถ้ายังไม่ได้ทำ)
  2. Vercel Dashboard → Add New → Project
  3. เลือก Repository จาก GitHub → กด Import
  4. ตั้งค่า Framework / Env Vars → กด Deploy
  5. เสร็จ! จากนี้ทุกครั้งที่ Push ไป main → Production อัปเดตอัตโนมัติ

🌿 Branch กับ Deployment

Push ไป BranchVercel ทำอะไรURL
main🌍 Production Deployyour-site.vercel.app
Branch อื่น / PR👁️ Preview DeployURL ชั่วคราวสำหรับทดสอบ
💜 ขั้นตอนทำงานที่แนะนำ

1) สร้าง Branch ใหม่ → 2) แก้โค้ด → 3) Push → 4) Vercel สร้าง Preview URL ให้ทดสอบ → 5) ทดสอบแล้วโอเค → 6) Merge เข้า main → 7) Production อัปเดต!

🧠 ทดสอบความเข้าใจ
Push ไป main กับ Push ไป branch อื่น ต่างกันอย่างไรบน Vercel?
👆 กดเพื่อดูคำตอบ
🌍 main → Production Deploy (เว็บจริง) / 🔀 branch อื่น → Preview Deploy (URL ชั่วคราวสำหรับทดสอบ)
auto-deploy หมายความว่าอะไร?
👆 กดเพื่อดูคำตอบ
🤖 Vercel จะ Build + Deploy อัตโนมัติทุกครั้งที่เรา Push โค้ดไป GitHub ไม่ต้องกดปุ่มเอง!
