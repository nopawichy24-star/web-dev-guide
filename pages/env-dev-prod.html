🔐 Environment Variables แต่ละตัว

ตัวแปรเปิดเผยได้?ทำไม
NEXT_PUBLIC_SUPABASE_URL✅ ได้เป็นแค่ที่อยู่ API ไม่ใช่ Secret
NEXT_PUBLIC_SUPABASE_ANON_KEY🟡 ได้ถ้ามี RLSRLS จะป้องกันข้อมูลให้
SUPABASE_SERVICE_ROLE_KEY🔴 ห้ามเด็ดขาด!ข้าม RLS ได้ → เข้าถึงทุกอย่าง
🚨 ทำไมห้าม Push .env.local ขึ้น GitHub

ไฟล์ .env.local มี Secret จริง — ถ้า Push ขึ้น GitHub (โดยเฉพาะ Public repo) ใครก็เห็นได้ → ต้องเปลี่ยน Key ทันที!

✅ Best Practice

ทดสอบใน localhost จนมั่นใจแล้วค่อย Push → Vercel จะ Deploy ให้อัตโนมัติ อย่าแก้ Production โดยตรง!
