Debugging — หาและแก้บั๊ก

🟡 Intermediate ⏱️ อ่าน ~15 นาที
📋 Prerequisite
✓ JavaScript พื้นฐาน
✓ เคยรัน npm run dev
✓ เคยใช้ Browser

🔍 เครื่องมือ Debug ที่ต้องรู้

เครื่องมือเปิดอย่างไรใช้ทำอะไร
💻 ConsoleF12 → แท็บ Consoleดู Error, console.log, warning
📡 NetworkF12 → แท็บ Networkดู API calls, status code, response
🔧 ElementsF12 → แท็บ Elementsตรวจ HTML/CSS, แก้สดได้
📦 ApplicationF12 → แท็บ Applicationดู cookies, localStorage, session
📂 SourcesF12 → แท็บ Sourcesตั้ง breakpoint, debug ทีละบรรทัด

📊 ประเภท Error ที่พบบ่อย

ประเภทตัวอย่างสาเหตุวิธีแก้
Build ErrorModule not foundลืม install หรือ path ผิดnpm install + ตรวจ import path
Runtime ErrorCannot read undefinedข้อมูลยังไม่โหลดเพิ่ม if ตรวจก่อนใช้ หรือใช้ ?.
Syntax ErrorUnexpected tokenพิมพ์ผิด ลืม , ; }ดูบรรทัดที่ Error บอก
Reference Errorx is not definedตัวแปรยังไม่ประกาศตรวจชื่อ + ตรวจ import
Type Errorx is not a functionเรียกใช้สิ่งที่ไม่ใช่ฟังก์ชันตรวจว่า import ถูกตัว
Network ErrorFailed to fetchAPI ล่ม, URL ผิด, CORSตรวจ URL, Network Tab
Database ErrorRLS: insufficientRLS บล็อกตรวจ Policy + auth.uid()
Auth ErrorInvalid loginemail/password ผิดตรวจ credentials + Redirect URL

📝 console.log — เครื่องมือ Debug ง่ายที่สุด

ts📄 ใส่ในโค้ดเพื่อตรวจค่า
// ตรวจว่าตัวแปรมีค่าอะไร
console.log('user:', user)

// ตรวจว่า function ถูกเรียกไหม
console.log('=== handleSubmit ถูกเรียกแล้ว ===')

// ตรวจ response จาก Supabase
const { data, error } = await supabase.from('tasks').select('*')
console.log('data:', data)
console.log('error:', error)

// ⚠️ อย่าลืมลบ console.log ก่อน Deploy!

📡 Supabase Logs

Log ประเภทดูที่ไหนใช้ตรวจอะไร
Auth LogsSupabase → Logs → AuthLogin ล้มเหลว, Token หมดอายุ
Database LogsSupabase → Logs → PostgresQuery ช้า, RLS บล็อก
API LogsSupabase → Logs → APIRequest ที่เข้ามา, status code

▲ Vercel Logs

Log ประเภทดูที่ไหนใช้ตรวจอะไร
Build LogsDeployments → คลิก deploymentBuild failed, TypeScript error
Runtime LogsLogs → RuntimeAPI route error, Server error
Function LogsLogs → FunctionsAPI function timeout, error
