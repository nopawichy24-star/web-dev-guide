🧠 ทดสอบคำศัพท์
CRUD ย่อมาจากอะไร?
👆 กดเพื่อดูคำตอบ
Create (สร้าง), Read (อ่าน), Update (แก้ไข), Delete (ลบ) — 4 การดำเนินการพื้นฐานกับข้อมูล
RLS ย่อมาจากอะไร ทำหน้าที่อะไร?
👆 กดเพื่อดูคำตอบ
Row Level Security — กำหนดว่าใครเข้าถึงข้อมูลแถวไหนได้ เช่น ผู้ใช้เห็นเฉพาะ Task ของตัวเอง
BaaS คืออะไร?
👆 กดเพื่อดูคำตอบ
Backend-as-a-Service — บริการที่ให้ Backend สำเร็จรูป ไม่ต้องเขียน Server เอง เช่น Supabase
DNS คืออะไร?
👆 กดเพื่อดูคำตอบ
Domain Name System — แปลงชื่อเว็บ (เช่น google.com) เป็นที่อยู่ IP ที่คอมพิวเตอร์เข้าใจ
