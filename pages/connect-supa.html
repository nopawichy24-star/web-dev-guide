เชื่อมเว็บไซต์กับ Supabase

🟢 Beginner ⏱️ อ่าน ~10 นาที
📋 Prerequisite
✓ Supabase พื้นฐาน
✓ JavaScript

บทนี้จะสอนวิธีเชื่อมต่อโค้ด JavaScript กับ Supabase เพื่ออ่าน-เขียนข้อมูลจากเว็บไซต์

📦 ติดตั้ง Library

bash💻 Terminal — ในโฟลเดอร์โปรเจกต์
npm install @supabase/supabase-js

🔧 สร้าง Supabase Client

ts📄 lib/supabase.ts
import { createClient } from '@supabase/supabase-js'

const supabaseUrl = process.env.NEXT_PUBLIC_SUPABASE_URL!
const supabaseAnonKey = process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY!

export const supabase = createClient(supabaseUrl, supabaseAnonKey)
💡 ทำไมต้องใช้ Environment Variables?

เพราะ URL และ Key จะแตกต่างกันระหว่างเครื่องเรา (localhost) กับเว็บจริง (Production) และถ้าใส่ในโค้ดตรงๆ เวลา Push ขึ้น GitHub จะเปิดเผย Key! ดูรายละเอียดในบท 🔐 Environment Variables

✏️ Create — เพิ่มข้อมูล

ts📄 ตัวอย่างโค้ดใน Component
async function createTask(title: string) {
  // ดึง user ปัจจุบัน
  const { data: { user } } = await supabase.auth.getUser()
  if (!user) throw new Error('ยังไม่ได้เข้าสู่ระบบ')

  const { data, error } = await supabase
    .from('tasks')
    .insert({ title, user_id: user.id })
    .select()        // ให้ส่งข้อมูลที่สร้างกลับมา
    .single()        // รับแค่ 1 แถว

  if (error) {
    console.error('สร้าง Task ไม่สำเร็จ:', error.message)
    return null
  }
  return data
}

📖 Read — อ่านข้อมูล

ts📄 ตัวอย่างโค้ด
async function getTasks() {
  const { data, error } = await supabase
    .from('tasks')
    .select('*')
    .order('created_at', { ascending: false })

  if (error) {
    console.error('โหลด Tasks ไม่สำเร็จ:', error.message)
    return []
  }
  return data     // RLS จะกรองให้เห็นเฉพาะ Task ของ user นี้อัตโนมัติ!
}

✏️ Update — แก้ไขข้อมูล

ts📄 ตัวอย่างโค้ด
async function toggleTask(id: string, completed: boolean) {
  const { error } = await supabase
    .from('tasks')
    .update({ completed: !completed })
    .eq('id', id)              // WHERE id = ...

  if (error) {
    console.error('แก้ไข Task ไม่สำเร็จ:', error.message)
  }
}

🗑️ Delete — ลบข้อมูล

ts📄 ตัวอย่างโค้ด
async function deleteTask(id: string) {
  // แนะนำ: ถามยืนยันก่อนลบเสมอ!
  const confirmed = window.confirm('ต้องการลบ Task นี้จริงหรือ?')
  if (!confirmed) return

  const { error } = await supabase
    .from('tasks')
    .delete()
    .eq('id', id)

  if (error) {
    console.error('ลบ Task ไม่สำเร็จ:', error.message)
  }
}
