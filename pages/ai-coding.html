📋 Cheat Sheet — คำสั่งที่ใช้บ่อย

📦 Git

bash💻 คำสั่ง Git ที่ใช้ทุกวัน
git clone URL             # ดาวน์โหลดโปรเจกต์
git status                # ดูสถานะไฟล์
git add .                 # เพิ่มทุกไฟล์เข้า Staging
git commit -m "message"   # บันทึก
git pull                  # ดึงโค้ดล่าสุด
git push                  # ส่งโค้ดขึ้น GitHub
git log --oneline         # ดูประวัติ Commit
git switch -c branch-name # สร้าง+สลับ Branch

📦 npm

bash💻 คำสั่ง npm ที่ใช้บ่อย
npm install               # ติดตั้ง dependencies ทั้งหมด
npm install package-name  # ติดตั้ง package เพิ่ม
npm run dev               # รันเว็บใน localhost
npm run build             # สร้างเว็บสำหรับ production
npm run start             # รัน production build

⚡ Supabase JS

ts📄 คำสั่ง Supabase ที่ใช้บ่อย
// อ่านทั้งหมด
supabase.from('tasks').select('*')

// อ่านแบบกรอง
supabase.from('tasks').select('*').eq('completed', true)

// เพิ่ม
supabase.from('tasks').insert({ title: '...', user_id: '...' })

// แก้
supabase.from('tasks').update({ completed: true }).eq('id', '...')

// ลบ
supabase.from('tasks').delete().eq('id', '...')

// Login
supabase.auth.signInWithPassword({ email, password })

// Signup
supabase.auth.signUp({ email, password })

// Logout
supabase.auth.signOut()

// ดู User
supabase.auth.getUser()

  • ❌ ห้ามวางข้อมูลส่วนตัวของผู้ใช้จริง
