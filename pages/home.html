เรียนรู้การสร้างเว็บไซต์ตั้งแต่เริ่มต้นจนเปิดใช้งานจริง ด้วย Git, GitHub, Vercel และ Supabase

💡 คู่มือนี้เหมาะกับใคร?
  • ผู้เริ่มต้นสร้างเว็บไซต์ ที่ยังไม่มีพื้นฐานมากนัก
  • ผู้ที่ใช้ AI เช่น Claude หรือ ChatGPT ช่วยเขียนโค้ด
  • ผู้ที่ต้องการ Deploy เว็บจริงโดยไม่ต้องตั้ง Server เอง

🧩 แต่ละเครื่องมือทำหน้าที่อะไร?

เครื่องมือหน้าที่เปรียบเทียบ
Git ติดตามการเปลี่ยนแปลงของโค้ด 📸 เหมือน "ประวัติการแก้ไข" ใน Google Docs — ย้อนดูได้ทุกเวอร์ชัน
GitHub เก็บโค้ดบนคลาวด์ ☁️ เหมือน Google Drive แต่สำหรับโค้ดโดยเฉพาะ
Vercel แปลงโค้ดเป็นเว็บที่เปิดได้ 🏗️ เหมือนช่างก่อสร้างที่สร้างบ้านจากแบบแปลน
Supabase ฐานข้อมูล + ระบบสมาชิก 🗄️ เหมือนตู้เก็บเอกสารอัจฉริยะ ที่จัดการสิทธิ์ให้ด้วย

📚 เส้นทางการเรียน

คู่มือนี้แบ่งเป็น 6 ระยะ เรียนตามลำดับจะเข้าใจง่ายที่สุด:

ระยะหัวข้อสิ่งที่จะทำได้
1️⃣พื้นฐานเข้าใจโครงสร้างเว็บ, ใช้ Command Line ได้, ติดตั้งเครื่องมือ
2️⃣Gitควบคุมเวอร์ชันของโค้ด, บันทึกและย้อนกลับได้
3️⃣GitHubเก็บโค้ดบนคลาวด์, ทำงานร่วมกับทีม
4️⃣Vercelเผยแพร่เว็บไซต์ขึ้นออนไลน์
5️⃣Supabaseสร้างฐานข้อมูล, ระบบสมาชิก, จัดการไฟล์
6️⃣โปรเจกต์จริงสร้าง Task App ครบวงจร + แก้ปัญหาได้
💜 เคล็ดลับการเรียน

กดคำภาษาอังกฤษที่ highlight ไว้เพื่อดูคำแปลไทย ทำตามทีละขั้นตอน อ่านกล่องคำเตือนทุกกล่อง แล้วลองทำแบบทดสอบท้ายบทเพื่อทบทวน!
