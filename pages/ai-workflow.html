AI Workflow — ทำงานร่วมกับ AI

🟡 Intermediate ⏱️ อ่าน ~12 นาที
📋 Prerequisite
✓ Git + GitHub พื้นฐาน
✓ Supabase พื้นฐาน
✓ เคยใช้ Claude หรือ ChatGPT

🔄 Workflow เต็มรูปแบบ

1️⃣ 💡 คิด Feature
ต้องการอะไร? ผู้ใช้จะได้อะไร?
⬇️
2️⃣ 📝 เขียน Requirement
Framework? Database? ข้อจำกัด?
⬇️
3️⃣ 🤖 ให้ AI วางแผน
"ช่วยวาง Architecture / ไฟล์ที่ต้องสร้าง"
⬇️
4️⃣ 💻 ให้ AI เขียน Code
ให้เขียนทีละไฟล์ ไม่ใช่ทีเดียวทั้งหมด
⬇️
5️⃣ 👀 Review
อ่านโค้ดทุกบรรทัด! ตรวจ Security
⬇️
6️⃣ 🐛 Debug
รันใน localhost → แก้ Error
⬇️
7️⃣ 🧪 Test
ทดสอบทุก flow: สมัคร, login, CRUD
⬇️
8️⃣ 💾 Commit → Push → Deploy!
เว็บอัปเดตอัตโนมัติ 🎉

✅ Prompt ที่ดี vs ❌ Prompt ที่ไม่ดี

❌ ไม่ดี

"ทำ Login"

← ไม่บอก Framework, Database, ภาษา, ข้อกำหนด อะไรเลย

✅ ดี

"สร้างระบบ Login ด้วย Supabase รองรับ Email Login ใช้ Next.js App Router แบ่ง Component ใช้ TypeScript"

📋 Checklist ก่อนถาม AI

❌ Common Mistake

ขอ AI เขียนทั้งโปรเจกต์ในครั้งเดียว → โค้ดจะยาวเกิน ผิดพลาดเยอะ ไม่เข้าใจ — ขอทีละฟีเจอร์ ทีละไฟล์

🧠 ทดสอบความเข้าใจ
ก่อนให้ AI เขียนโค้ด ควรทำอะไรก่อน?
👆 กดเพื่อดูคำตอบ
📝 เขียน Requirement ให้ชัด: Framework, Database, ภาษา, ข้อกำหนด → แล้วให้ AI วางแผนก่อนเขียนโค้ด
หลัง AI เขียนโค้ดให้แล้ว ต้องทำอะไร?
👆 กดเพื่อดูคำตอบ
👀 Review → 🐛 Debug → 🧪 Test → 💾 Commit อย่าใช้โค้ดโดยไม่อ่านเด็ดขาด!
