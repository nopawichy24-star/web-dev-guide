🔧

ติดตั้งเครื่องมือ

🟢 Beginner ⏱️ อ่าน ~10 นาที
📋 Prerequisite
✓ พื้นฐาน Command Line

ก่อนเริ่มพัฒนาเว็บ เราต้องติดตั้งเครื่องมือ 3 อย่างนี้:

1️⃣ ติดตั้ง Node.js

Node.js ทำให้คอมพิวเตอร์ของเราสามารถรันโค้ด JavaScript ได้ และมาพร้อมกับ npm

📥 วิธีติดตั้ง

  1. เข้าเว็บ nodejs.org
  2. กดดาวน์โหลดเวอร์ชัน LTS (Long Term Support — เวอร์ชันที่เสถียร)
  3. เปิดไฟล์ที่ดาวน์โหลดมา แล้วกด Next/ตกลง ไปเรื่อยๆ จนเสร็จ

✅ ตรวจสอบว่าติดตั้งสำเร็จ

เปิด Terminal แล้วพิมพ์:

bash 💻 Terminal 
node --version
npm --version

ผลลัพธ์ที่ควรเห็น: (ตัวเลขอาจต่างกันเล็กน้อย)

output
v22.15.0
10.9.2
⚠️ ถ้าขึ้นว่า "command not found"

หมายความว่ายังไม่ได้ติดตั้ง หรือติดตั้งไม่สำเร็จ ให้ลองปิด Terminal แล้วเปิดใหม่ ถ้ายังไม่ได้ ให้ดาวน์โหลดและติดตั้งใหม่อีกครั้ง

2️⃣ ติดตั้ง Git

📥 วิธีติดตั้ง

ระบบวิธีติดตั้ง
🪟 Windowsดาวน์โหลดจาก git-scm.com → ติดตั้ง → กด Next ไปเรื่อยๆ (ใช้ค่าเริ่มต้นได้เลย)
🍎 Macเปิด Terminal พิมพ์ git --version → ถ้ายังไม่มี ระบบจะถามให้ติดตั้ง → กดตกลง

✅ ตรวจสอบว่าติดตั้งสำเร็จ

bash 💻 Terminal 
git --version

ผลลัพธ์ที่ควรเห็น:

output
git version 2.47.1

⚙️ ตั้งค่า Git (ทำครั้งเดียว)

หลังติดตั้ง Git แล้ว ต้องบอก Git ว่าเราชื่ออะไร ใช้อีเมลอะไร:

bash 💻 Terminal 
# เปลี่ยน "ชื่อของคุณ" เป็นชื่อจริงของคุณ
git config --global user.name "ชื่อของคุณ"

# เปลี่ยนเป็นอีเมลที่ใช้สมัคร GitHub
git config --global user.email "your-email@example.com"
💡 อธิบาย

--global หมายถึงตั้งค่าสำหรับทุกโปรเจกต์ในเครื่องนี้ ข้อมูลนี้จะแสดงใน commit history ของ Git ไม่ใช่รหัสผ่าน!

✅ ตรวจสอบค่าที่ตั้ง

bash 💻 Terminal 
git config --global user.name
git config --global user.email

ถ้าแสดงชื่อและอีเมลที่เราตั้งไว้ = สำเร็จ! ✅

3️⃣ ติดตั้ง VS Code

📥 วิธีติดตั้ง

  1. เข้าเว็บ code.visualstudio.com
  2. กดดาวน์โหลดตามระบบปฏิบัติการ
  3. ติดตั้งตามขั้นตอน

🔌 Extensions ที่แนะนำ

เปิด VS Code → กดไอคอน Extensions ที่ Sidebar ซ้าย (หรือกด Ctrl+Shift+X) → ค้นหาและติดตั้ง:

Extensionทำอะไร
ESLintตรวจสอบโค้ด JavaScript/TypeScript
Prettierจัดรูปแบบโค้ดอัตโนมัติ
Live Serverดูผลเว็บไซต์แบบ real-time
GitLensดูประวัติ Git ใน VS Code
