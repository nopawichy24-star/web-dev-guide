พื้นฐาน Command Line

🟢 Beginner ⏱️ อ่าน ~8 นาที
📋 Prerequisite
✓ พื้นฐานการใช้คอมพิวเตอร์

Command Line (หรือ Terminal) คือเครื่องมือที่เราพิมพ์คำสั่งเพื่อสั่งคอมพิวเตอร์ แทนที่จะใช้เมาส์คลิก

💡 ทำไมต้องใช้ Command Line?

เครื่องมือพัฒนาเว็บสมัยใหม่ เช่น Git, npm, Node.js ล้วนใช้ผ่าน Command Line ทั้งนั้น ถ้าเราคุ้นเคยกับมัน จะทำงานได้เร็วขึ้นมาก!

🖥️ เปิด Terminal อย่างไร?

ระบบปฏิบัติการวิธีเปิด
🪟 Windowsกด Win + R พิมพ์ cmd แล้วกด Enter — หรือค้นหา "Terminal" ใน Start Menu
🍎 Macกด Cmd + Space พิมพ์ "Terminal" แล้วกด Enter
💻 VS Codeกด Ctrl + ` (backtick) — หรือเมนู Terminal → New Terminal
💜 แนะนำ

ใช้ Terminal ใน VS Code เพราะสะดวกที่สุด — เปิดโค้ดและ Terminal ในหน้าจอเดียวกัน!

📂 คำสั่งจัดการไฟล์และโฟลเดอร์

pwd — ดูว่าเราอยู่ที่ไหน

pwd = แสดงตำแหน่งโฟลเดอร์ที่เราอยู่ตอนนี้

bash 💻 Terminal 
pwd

ผลลัพธ์ตัวอย่าง:

output
/Users/nopawich/Documents

ls — ดูไฟล์ในโฟลเดอร์

ls = แสดงรายชื่อไฟล์และโฟลเดอร์ทั้งหมดที่อยู่ตรงนี้

bash 💻 Terminal 
# Mac / Linux
ls

# Windows (cmd)
dir

# Windows (PowerShell)
ls
💡 Windows vs Mac

คำสั่ง ls ใช้ได้ใน Mac, Linux และ PowerShell ของ Windows — ถ้าใช้ cmd ต้องใช้ dir แทน แต่ถ้าใช้ Terminal ใน VS Code จะใช้ ls ได้เลย

cd — เปลี่ยนโฟลเดอร์

cd = ย้ายไปที่โฟลเดอร์ที่ต้องการ

bash 💻 Terminal 
# เข้าไปในโฟลเดอร์ Documents
cd Documents

# ถอยกลับ 1 ระดับ
cd ..

# กลับไปที่โฟลเดอร์หลัก (home)
cd ~

# เข้าไปในโฟลเดอร์ my-project ที่อยู่ใน Desktop
cd Desktop/my-project
⚠️ ข้อควรระวัง

ชื่อโฟลเดอร์ต้องพิมพ์ให้ตรง! ถ้าชื่อมีช่องว่าง ต้องใส่เครื่องหมายคำพูดครอบ เช่น cd "My Project"

mkdir — สร้างโฟลเดอร์ใหม่

mkdir = สร้างโฟลเดอร์ใหม่

bash 💻 Terminal 
# สร้างโฟลเดอร์ชื่อ my-website
mkdir my-website

# เข้าไปในโฟลเดอร์ที่เพิ่งสร้าง
cd my-website

touch / echo — สร้างไฟล์ใหม่

bash 💻 Terminal 
# Mac / Linux — สร้างไฟล์เปล่า
touch index.html

# Windows (PowerShell) — สร้างไฟล์เปล่า
New-Item index.html

# ทุกระบบ — สร้างไฟล์พร้อมเนื้อหา
echo "Hello World" > index.html

คำสั่งอื่นที่มีประโยชน์

คำสั่งความหมายตัวอย่าง
catแสดงเนื้อหาไฟล์cat index.html
cpCopy คัดลอกไฟล์cp file1.txt file2.txt
mvMove ย้าย/เปลี่ยนชื่อmv old.txt new.txt
rmRemove ลบไฟล์rm file.txt
clearล้างหน้าจอ Terminalclear
🚨 อันตราย!

คำสั่ง rm ลบไฟล์แล้วกู้คืนไม่ได้! ไม่มีถังขยะ! ตรวจสอบชื่อไฟล์ให้แน่ใจก่อนลบทุกครั้ง อย่าใช้ rm -rf / เด็ดขาด (จะลบทุกอย่างในเครื่อง!)

📦 คำสั่ง npm ที่ต้องรู้

npm คือตัวจัดการ package (ไลบรารี่) ของ JavaScript ติดตั้งมาพร้อมกับ Node.js

bash 💻 Terminal 
# สร้างโปรเจกต์ใหม่ (สร้างไฟล์ package.json)
npm init -y

# ติดตั้ง package (เช่น supabase)
npm install @supabase/supabase-js

# รันโปรเจกต์ (ขึ้นอยู่กับคำสั่งในไฟล์ package.json)
npm run dev

# ตรวจสอบ version ของ npm
npm --version

👆 กดเพื่อดูคำตอบ

npm install ทำอะไร?
👆 กดเพื่อดูคำตอบ
📦 ติดตั้ง package (ไลบรารี่) เพิ่มเข้าในโปรเจกต์
ทำไม rm ถึงอันตราย?
👆 กดเพื่อดูคำตอบ
