Environment Variables

🟢 Beginner ⏱️ อ่าน ~8 นาที
📋 Prerequisite
✓ Supabase
✓ Vercel

Environment Variables คือค่าตั้งค่าที่เก็บแยกจากโค้ด — เช่น API Key, รหัสลับ ที่ไม่ควรใส่ในโค้ดตรงๆ

💡 ทำไมต้องแยก?

1) ความปลอดภัย — ไม่ต้อง Push Secret ขึ้น GitHub
2) ยืดหยุ่น — เครื่องเรา (localhost) กับเว็บจริง (Production) ใช้ค่าต่างกันได้

📁 ไฟล์ .env แต่ละชนิด

ไฟล์ใช้เมื่อไหร่ใส่ใน .gitignore?
.envค่าเริ่มต้นทั่วไป✅ ใช่
.env.localค่าสำหรับเครื่องเราเอง (ใช้บ่อยสุด!)✅ ใช่
.env.exampleตัวอย่าง — ไม่มี Secret จริง ไว้ให้คนอื่น Copy ไปตั้ง❌ Push ได้

ตัวอย่าง .env.local

env📄 .env.local — ไฟล์นี้ห้าม Push!
# Supabase (ใส่ค่าจริงจาก Dashboard)
NEXT_PUBLIC_SUPABASE_URL=https://xxxxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIs...

# Secret — ห้ามมี NEXT_PUBLIC_ !!!
SUPABASE_SERVICE_ROLE_KEY=eyJhbGciOiJIUzI1NiIs...

ตัวอย่าง .env.example

env📄 .env.example — Push ขึ้น GitHub ได้
# คัดลอกไฟล์นี้เป็น .env.local แล้วใส่ค่าจริง
# cp .env.example .env.local

NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=

🔓 Public vs Private Variable

🔓 Public

ขึ้นต้นด้วย NEXT_PUBLIC_

ใช้ใน Browser ได้ ✅

ผู้ใช้เห็นได้! ⚠️

เช่น NEXT_PUBLIC_SUPABASE_URL

🔒 Private

ไม่มี prefix NEXT_PUBLIC_

ใช้ได้เฉพาะ Server ❌

ผู้ใช้เห็นไม่ได้ ✅

เช่น SUPABASE_SERVICE_ROLE_KEY

🚨 กฎเหล็ก!

Secret ต้องไม่มี NEXT_PUBLIC_ — ถ้าตั้งชื่อว่า NEXT_PUBLIC_SUPABASE_SERVICE_ROLE_KEY ค่านี้จะถูกส่งไป Browser ทุกคนเห็นได้ → อันตรายมาก!

⚙️ ตั้ง Env Vars บน Vercel

  1. Vercel Dashboard → เลือกโปรเจกต์ → Settings → Environment Variables
  2. เพิ่มตัวแปรทีละตัว (ชื่อ + ค่า)
  3. เลือกว่าใช้ใน Environment ไหน (Production / Preview / Development)
  4. กด Save
  5. Redeploy เว็บ! (เปลี่ยน Env Vars แล้วต้อง Deploy ใหม่ถึงจะมีผล)

🔧 ข้อผิดพลาดที่พบบ่อย

ปัญหาสาเหตุวิธีแก้
ค่าเป็น undefinedชื่อตัวแปรผิด หรือลืมเพิ่มใน Vercelตรวจชื่อให้ตรง ตรวจ prefix
เปลี่ยนค่าแล้วไม่มีผลลืม Redeployเปลี่ยน Env Vars แล้วต้อง Deploy ใหม่เสมอ!
ใช้ได้ใน Local แต่ Production ไม่ได้ค่าใน .env.local กับ Vercel ไม่ตรงกันตรวจค่าให้ตรงกันทั้ง 2 ที่
มี "" หรือช่องว่างใส่เครื่องหมายคำพูดหรือเว้นวรรคในค่าค่าใน .env ไม่ต้องใส่ "" ครอบ
Secret อยู่ใน Browserใช้ prefix NEXT_PUBLIC_ กับ Secretเอา prefix ออก → ใช้ได้เฉพาะ Server
